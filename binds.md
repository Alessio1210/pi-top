# Hardware Binding Konfiguration (Foundation Plate)

1. **Tür-Sensor**
   - Typ: WiFi
   - Status: Verbunden via Netzwerk

2. **Lautsprecher / Summer**
   - Port: **A2**
   - Modus: Digital Output

3. **Touch Keypad (für PIN Eingabe)**
   - Typ: Grove I2C Touch Sensor (MPR121) oder ähnlich
   - Port: **I2C** (Da ein Keypad zu komplex für Port D1 ist, muss es I2C sein, außer es ist USB)

4. **LCD Display (16x2)**
   - Port: **I2C**
